4.2 Linear Regression of an Indicator MatrixÂ¶

Suppose \(\mathcal{G}\) has \(K\) classes, each response category is associated with an indicator \(Y_k\) with \(Y_k = 1\) if \(G = k\) else \(0\). These are collected together in a vector \(Y = (Y_1, \dots, Y_k)\), and the \(N\) training instances of these form an \(N \times K\) indicator response matrix \(\mathbf{Y}\). We fit a regression model to each of the columns of \(\mathbf{Y}\) simultaneously, and the fit is given by

\[\hat{\mathbf{Y}} = \mathbf{X}(\mathbf{X}^\top \mathbf{X})^{-1}\mathbf{X}^\top\mathbf{Y}\]

From which we obtain the \((p+1) \times K\) coefficient matrix \(\hat{\mathbf{B}} = (\mathbf{X}^\top \mathbf{X})^{-1}\mathbf{X}^\top\mathbf{Y}\). A new observation with input \(x\) is classified by first computing the fitted output \(\hat{f}(x)^\top = (1, x^\top)\hat{\mathbf{B}}\) and then predicting the category by identifying the largest component: \(\hat{G}(x) = \text{argmax}_{k \in \mathcal{G}}\hat{f}_k(x)\).
